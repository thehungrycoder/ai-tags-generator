AI Tags Generator is a WordPress plugin that uses OpenAI to generate tags for your blog posts.

== Description ==

AI Tags Generator is a WordPress plugin that uses OpenAI to generate SEO-friendly tags for your blog posts based on their content and category. Simply enter your OpenAI API Key on the plugin's settings page, and the plugin will automatically generate tags when you create or update a post. You can also manually generate tags for a post from the post list.

== Installation ==

1. Upload the `ai-tags-generator` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your OpenAI API Key on the AI Tags Generator settings page.

== Usage ==

To use the AI Tags Generator plugin, simply create or update a post as usual. The plugin will automatically generate tags based on the post's content and category.

If you want to manually generate tags for a post, go to the post list in the WordPress admin, and find the 'Generate Tags' action in the row of actions for the post. Click this action to generate tags for the post.

Remember to enter your OpenAI API Key on the AI Tags Generator settings page before using the plugin.
